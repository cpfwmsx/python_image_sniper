# python图片爬虫

### 介绍
使用Python做网络资源抓取。<br>
郑重申明，本项目仅做学习使用，勿做他用。<br>
项目中的网络抓取技术非针对全站进行爬取，不可直接用于生产。<br>


####  目前抓取内容说明
为保持程序活跃度，抓取的网站都是好玩的文字或者图片，目标站点如下：
